Week Day Summary

This plugin is created for get the summery of week days based on the selection result will be if:
All day selected: it will return 'All Day',
Monday to Friday selected: it will return 'Week Days',
Saturday & Sunday selected: it will return 'Weekend',
Wednesday, Thursday, Friday selected: it will return 'Wednesday - Friday'
